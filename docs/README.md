# M31 Stability Analysis Documentation

This directory contains documentation for the M31 (Andromeda Galaxy) stability analysis project.

## Overview

The M31 Stability project performs N-body simulations of tracer particles in the Andromeda Galaxy to study disk dynamics and stability under a flat rotation curve.

## Key Features

- Leapfrog integration scheme
- Logarithmic potential model
- Flat rotation curve (v_c = 220 km/s)
- 500 tracer particles
- 1 Gyr evolution time
